# Template construction

After preprocessing (`01_preprocessing/`), each age×sex cohort is built into an unbiased
group-average template with the ANTs multivariate template-construction workflow
(`antsMultivariateTemplateConstruction2.sh`, ANTs ≥ 2.5).

Two forms are provided:

- **`mri_template_construction.sh <age> <sex>`** — generalized, parameterized driver
  (recommended for reuse). Reads `DB` / `C_REG` / `TMPROOT` / `NT` from the environment
  or `../config.sh`.
- **`as_run/mri-template-creation-age{5..18}{m,f}.sh`** — the 30 exact as-run scripts
  that produced the published templates (28 strata + the two age-16 low-N variants),
  preserved for provenance. They are identical except for the stratum and the single
  seed subject; subject identifiers have been replaced with `REFERENCE_SUBJECT`.

## As-run invocation (identical across all 28 published strata)

```bash
antsMultivariateTemplateConstruction2.sh \
  -d 3 -o template0_ -i 12 \           # 3D, 12 template-building iterations
  -z REFERENCE_SUBJECT.nii.gz -r 1 \   # seed with one in-stratum scan, rigid pre-align
  -g 0.10 -c 0 -k 1 -w 1 -y 0 \        # gradient step 0.10; -c 0 = run locally (no scheduler)
  -f 8x4x2x1 -s 4x2x1x0vox \           # 4-level shrink / smoothing schedule
  -q 120x90x60x20 \                    # per-level max iterations
  -l 0 \                               # no histogram matching between images
  -t "SyN[0.1]" -m "CC[2]" \           # SyN transform; cross-correlation metric, radius 2
  -n 0 -u 1 \                          # -n 0: NO per-iteration N4 (preserve native contrast)
  /path/to/stratum/*.nii.gz            # all preprocessed, selected inputs for the stratum
```

Key pediatric-specific choices:
- **Transform `SyN[0.1]`, metric `CC[2]`, gradient step `0.10`** — the values used for every
  released template. (Earlier *pilot* runs — `mri-template-creation-sample-*.sh`, not part of
  this release — explored `BSplineSyN` with gradient step `0.25`; those parameters do **not**
  correspond to the published templates.)
- **`-n 0`** disables per-iteration N4 intensity rescaling, preserving the CSF-anchored native
  tissue contrast established in preprocessing.
- **Seed (`-z`)** = a single high-quality in-stratum scan chosen during visual QC (not a blurred
  average), to accelerate convergence toward sharp consensus anatomy. The exact seed subject per
  stratum is recorded in the template **data deposit** manifest, not in this code repo.

Post-processing: residual midline tilt corrected by a rigid transform (ITK-SNAP) applied with
`antsApplyTransforms`; tissue-probability maps generated by warping each subject's tissue
segmentation into template space and averaging. See the methods paper for full justification.
